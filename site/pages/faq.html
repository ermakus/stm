FAQ

  • How to open downloaded file?

    You need to use the separate application. For a movies, I recommend yxflash player from the Cydia. Works perfectly for me.

  • How to delete downloaded file?

    Swipe on file name in "Save To" tab, then tap "Delete"

  • Which firmware versions is supported?

    I've tested it on 3.2.x and 4.0.2. iOS 4.1 is not supported yet.

  • Where is downloaded files located?

    By default, it stored in /var/mobile/Library/Downloads. You can change this location on "Save To" tab

  • What is .itorrent or .fastresume dir?

    This directory contain various temporary files. You can delete it safetely after torrent is downloaded.

  • Where to find torrents?

    Dude, just google it.
